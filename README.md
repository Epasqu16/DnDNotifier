# 🔔 DnDNotifier (Vencord Plugin)

[TR] Discord'da "Rahatsız Etmeyin" (DND) modundayken gelen doğrudan mesajlar (DM) için özel sesli bildirimler almanızı sağlayan gelişmiş bir Vencord eklentisidir.

[EN] An advanced Vencord plugin that allows you to receive custom audio notifications for Direct Messages (DMs) while in "Do Not Disturb" (DND) mode on Discord.

---

## ✨ Özellikler / Features

### 🇹🇷 Türkçe
- **Özel Arayüz:** Vencord ayarları içinde tamamen React ile tasarlanmış şık ve modern panel.
- **Ses Kaynağı:** Kendi ses dosyalarınızı bilgisayardan yükleyebilir veya doğrudan bir MP3/WAV linki kullanabilirsiniz.
- **Ses Seviyesi Kontrolü:** Bildirim sesini istediğiniz seviyeye ayarlayabilmeniz için entegre slider.
- **Hızlı Test:** Ayarlar panelinden sesinizi anında test edebilme butonu.
- **Akıllı Filtreleme:** Kendi mesajlarınız veya sunucu mesajları için gereksiz tetiklemeyi önler.

### 🇺🇸 English
- **Custom UI:** A sleek and modern settings panel designed entirely with React.
- **Sound Source:** Upload your own audio files from your computer or use a direct MP3/WAV URL.
- **Volume Control:** Integrated slider to adjust the notification volume to your preference.
- **Quick Test:** A dedicated button to test your sound instantly within the settings panel.
- **Smart Filtering:** Prevents unnecessary triggers for your own messages or guild (server) messages.

---

## 🛠️ Kurulum / Installation

### 🇹🇷 Nasıl Kurulur?
1. Vencord kaynak kodunu bilgisayarınıza indirin (`git clone`).
2. Bu depodaki `DnDNotifier` klasörünü Vencord projenizin içindeki `src/plugins/` klasörüne kopyalayın.
3. Terminalde `pnpm build` komutunu çalıştırarak eklentiyi derleyin.
4. Discord'u yeniden başlatın ve Vencord ayarlarından eklentiyi aktif edin.

### 🇺🇸 How to Install?
1. Clone the Vencord source code to your computer (`git clone`).
2. Copy the `DnDNotifier` folder from this repository into the `src/plugins/` folder of your Vencord project.
3. Run the `pnpm build` command in your terminal to compile the plugin.
4. Restart Discord and enable the plugin from Vencord settings.
